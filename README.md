# 🎓 Student Marks Analyzer (GUI Project)

A professional GUI-based Student Marks Analyzer built using Python.

## 🚀 Features
- Student Name & Roll Number Input
- Total, Average, Highest, Lowest Calculation
- Grade & Pass/Fail Detection
- Percentage Calculation
- 📊 Bar Chart Visualization (Matplotlib)
- 💾 Save Report as Text File
- Error Handling
- Clean UI Design

## 🛠 Tech Stack
- Python
- Tkinter
- Matplotlib

## 📂 How to Run

1. Install required library:
   pip install matplotlib

2. Run the application:
   python student_marks_analyzer.py
